Менеджер задач:
Суть проекта чтобы на практике понять как работают методы POST, GET, PUT, DELETE.

Использовал:
Python 3.12
FastAPI
Pydantic
Сервер uvicorn

Функции:
1)Создание задач
2)Просмотр всех задач
3)Обновление статуса
4)Удаление задачи 

В качестве базы данных я использовал обычный словарь (dict) 
который встроен в язык программирования Python

Данный проект я делал на пару с ИИ, но ИИ выступал в роли
наставника, логика польностью написана мной, ИИ всего лишь 
направлял меня


Подключил менеджер к серверу, отклик возможен долгий т.к сервер бесплатный.

Во всех случаях: 
1) сначала жмёте Try it out
2) Заполняете поля
3) Жмёте Execute
4) Ниже выводится результат

Если прокрутите вниз должно появиться ответ в формате JSON:
{
  "id": 1,
  "title": "Изучить алгоритмы ",
  "status": false
}
Данные взяты с полей которые я заполнил своим вариантом.

ВАЖНО!
После 15 минут бездействия сервер "засыпает" и включается как только кто-то пытается зайти в форму,
если вы видите окно загрузки (черный экран с выводом статусов сервера) просто подождите 1 - 2 минуты,
сервер "проснётся" и пустит вас к форме.
