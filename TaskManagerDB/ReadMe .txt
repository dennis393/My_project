Менеджер задач:
Суть проекта чтобы на практике понять как работают методы POST, GET, PUT, DELETE.
Так же понять как подключить базу данных к backend' у.

Использовал:
Python 3.12
FastAPI
Pydantic
Сервер uvicorn
Базу данных SQLite

Функции:
1)Создание задач
2)Просмотр всех задач
3)Обновление статуса
4)Удаление задачи 

Базу данных я выбрал SQLite потому что она лёгкая, для неё не требуется отдельный сервер и она уже встроена в сам язык программирования Python. Статус выполнения какой-либо задачи записывается как 0 / 1, но FastAPI возвращает True / False

Проект подключен к серверу и работает, возможны задержки т.к проект подключен к бесплатному серверу. Если вы видите черный экран с выводом статуса сервера, положите 1 - 2 минуты сервер "проснётся" и пустит вас к форме.

Можете ознакомиться здесь:
https://my-project-j7m2.onrender.com/docs

ВАЖНО!
1) После 15 минут бездействия сервер "засыпает" и включается как только кто-то пытается зайти в форму, если вы видите окно загрузки (черный экран с выводом статусов сервера) просто подождите 1 - 2 минуты, сервер "проснётся" и пустит вас к форме.

2) Как только вы выйдете из swagger'а база данных очистится. Увы, с этим ничего не поделать.
